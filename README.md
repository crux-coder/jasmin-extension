# Jasmin's Extension

This extension will hide all the help on the Free Code Camp website so you can focus on testing yourself without being tempet to ask for help. Of course, if needed, extension can be disabled at any time.

## Legend

- FCC - Free Code Camp
- :construction: -> Comming soon
- :warning: -> Buggy (not a beach buggy)
- :white_check_mark: -> Fully implemented

## Features

- Hides all help from every Free Code Camp challenge :white_check_mark:
- Offers navigation between challenges :white_check_mark:
- Enable/Disable at any time :white_check_mark:
- Feedback when challenge on FCC when tests fail :construction:
- Progress tracking :construction:
- Time tracking :construction:
- Test mode :construction:
